Unique(ComplexDArrayList)

Returns a wrapper for the specified ComplexDArrayList ensuring that all elements are unique.

Declaration
public static ComplexDArrayList Unique(ComplexDArrayList collection)
Parameters
ComplexDArrayList collection

The ComplexDArrayList to wrap.
Returns
ComplexDArrayList

A wrapper around collection ensuring that all elements are unique.
Remarks

The Unique wrapper provides a set-like collection by ensuring that all elements in the ComplexDArrayList are unique.

Unique raises an System.ArgumentException if the specified collection contains any duplicate elements. The returned wrapper raises a System.NotSupportedException whenever the user attempts to add an element that is already contained in the ComplexDArrayList.

Note: The Unique wrapper reflects any changes made to the underlying collection, including the possible creation of duplicate elements. The uniqueness of all elements is therefore no longer assured if the underlying collection is manipulated directly.

Exceptions
System.ArgumentException

collection contains duplicate elements.
System.ArgumentNullException

collection is a null reference.
