Unique(Vector3DArrayList)

Returns a wrapper for the specified Vector3DArrayList ensuring that all elements are unique.

Declaration
public static Vector3DArrayList Unique(Vector3DArrayList collection)
Parameters
Vector3DArrayList collection

The Vector3DArrayList to wrap.
Returns
Vector3DArrayList

A wrapper around collection ensuring that all elements are unique.
Remarks

The Unique wrapper provides a set-like collection by ensuring that all elements in the Vector3DArrayList are unique.

Unique raises an System.ArgumentException if the specified collection contains any duplicate elements. The returned wrapper raises a System.NotSupportedException whenever the user attempts to add an element that is already contained in the Vector3DArrayList.

Note: The Unique wrapper reflects any changes made to the underlying collection, including the possible creation of duplicate elements. The uniqueness of all elements is therefore no longer assured if the underlying collection is manipulated directly.

Exceptions
System.ArgumentException

collection contains duplicate elements.
System.ArgumentNullException

collection is a null reference.
