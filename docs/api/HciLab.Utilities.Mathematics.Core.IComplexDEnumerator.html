Interface IComplexDEnumerator

Supports type-safe iteration over a collection that contains ComplexD elements.

Namespace: System.Dynamic.ExpandoObject
Assembly: HciLab.Utilities.dll
Syntax
public interface IComplexDEnumerator
Remarks

IComplexDEnumerator provides an System.Collections.IEnumerator that is strongly typed for ComplexD elements.

Properties

Current

Gets the current ComplexD element in the collection.

Declaration
ComplexD Current { get; }
Property Value
ComplexD

The current ComplexD element in the collection.
Remarks

Please refer to System.Collections.IEnumerator.Current for details, but note that Current fails if the collection was modified since the last successful call to MoveNext() or Reset().

Exceptions
System.InvalidOperationException

The enumerator is positioned before the first element of the collection or after the last element.

-or-

The collection was modified after the enumerator was created.
